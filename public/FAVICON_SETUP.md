# Favicon Setup

To complete the favicon setup, you'll need to:

1. Export the Promptr logo (98e8f0edad5a3c85110d1f971d685e0f49e9a80b.png) as:
   - favicon-32x32.png (32x32 pixels)
   - favicon-16x16.png (16x16 pixels)
   - og-image.png (1200x630 pixels for social media preview)

2. Place these files in the `/public` directory

The HTML file has already been configured to reference these favicons.
